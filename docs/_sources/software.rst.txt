Software vereisten
==================

Installatie en configuratie moeten gescript worden en in een Github repository worden beheerd. 
Het is daarmee ook beschikbaar voor andere CoderDojo’s.

OS
--

* Ubuntu Server 18.04LTS
* SSH server
* WIFI hotspot software

Router
------

* Firewall
* DHCP server
* DNS server

Webserver
---------

* Apache
* WordPress (portaal, installatieprogramma’s)
* GitBucket git webserver (instructies)

Docker
------

* Docker host
* Docker registry
* Portainer (web UI om de images/containers te managen of nieuwe images te maken)
